#### 【1】linux软件安装

​	 1.了解不同操作系统的软件包类型。点后缀

​	   windows      .exe

​	   android      .apk

​	   ubuntu       .deb

​	 2.ubuntu使用的是deb软件包

​	   deb软件包分为：

​	     二进制软件包：.deb   (安装)

​		 源码包：.dsc      （软件源码）

​		 sl_3.03-16_i386.deb

​		 软件名_版本号.次版本号-修订次数_架构.后缀

​	 3.软件包的管理工具：dpkg apt 

​	 1)dpkg

​	    特点：

​		  安装软件需要软件的安装包存在，不能从镜像站点获取软件包，不能检测软件之间的依赖关系，安装不需要网络。
